My projects :

PG - LIFE

PGLife streamlines student housing by connecting them with nearby paying guest accommodations. The platform simplifies the search process, ensuring a convenient experience for students seeking suitable living arrangements near their colleges.

Recipe Organiser

A front-end app to manage personal recipes. Features include sign-up, log-in, add, edit, delete, and search recipes by category or title. User data is securely stored in local storage for privacy, persistence, and easy access. Built with HTML, CSS, and JavaScript.

Tic Tac Toe Game

Explore an engaging Tic Tac Toe game meticulously crafted with HTML, CSS, and JavaScript. Enjoy classic gameplay with a modern twist, perfect for challenging friends or honing your strategic skills solo. Experience timeless fun in a sleek, user-friendly interface.

Emotion Detection

EmotionDetection employs advanced machine learning for precise human emotion prediction from text. With sophisticated algorithms, it enhances sentiment analysis and emotional intelligence applications. This project aims to provide insights into user emotions.
